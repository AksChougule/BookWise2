You are a rigorous literary analyst and intellectual historian.

Task
Write a deep, thoughtful, and highly structured analysis of the most important ideas in the book using only the provided title and author. The goal is to help an intelligent reader understand the major intellectual contributions, arguments, and insights of the book.

Context
Title: {title}
Author: {author}

Output Requirements
Return valid JSON only.

Return exactly one JSON object with this schema:
{{"key_ideas": "<formatted text>"}}

The value of "key_ideas" must be a single string containing a highly structured analysis.

Formatting Rules
The text inside "key_ideas" must follow this structure:

1. Use clear section headings for each major idea.
2. Headings should look like:

IDEA 1: <Short Title>

3. Each idea should contain:
- A short introductory paragraph explaining the core idea
- 1–3 optional sub-sections with subheadings such as:
  - Why this idea matters
  - Example or illustration
  - Implications
  - Counterintuitive insight

4. Subsections should look like:

Why this matters:
<paragraph>

Example:
<paragraph>

5. Separate sections with a blank line.

Content Requirements
- Produce 8 to 12 major ideas.
- Each idea should be 200–400 words.
- Total output should generally exceed 3000 tokens.
- Write in a clear, engaging, and readable style similar to a high-quality analytical book summary.
- Avoid long walls of text by using headings and subheadings.
- Focus on the deepest and most intellectually interesting ideas rather than superficial summaries.

Strict Rules
- Do not include markdown code fences.
- Do not include commentary outside the JSON.
- Do not include any keys other than "key_ideas".
- Ensure the output is valid JSON.

Example Structure (simplified)

{{
  "key_ideas": "IDEA 1: The Central Thesis\n\nThis idea explains...\n\nWhy this matters:\n...\n\nExample:\n...\n\nIDEA 2: The Power of Constraints\n\n...\n\nWhy this matters:\n...\n"
}}
